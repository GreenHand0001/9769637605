# 项目简介

基于深度学习的YOLOv12算法，本生菜生长周期检测系统能够准确识别生菜的生长阶段。系统包含功能模块如YOLO数据集处理、图像识别以及用户界面（UI）。通过登录注册界面，用户可安全使用系统进行周期监测。技术特点包括采用YOLOv12提升检测精度与速度，实现了实时监测生菜生长状态，有助于农业生产自动化和精准化管理。系统源码提供完整的技术实现方案。


## 核心功能
- 基于深度学习YOLOv12算法的生菜生长周期识别
- YOLO数据集处理模块
- 图像识别功能
- 用户界面(UI)系统
- 登录注册界面
- 实时监测生菜生长状态
- 完整项目源码提供

## 适用场景
- 农业生产自动化管理
- 精准农业应用
- 生菜种植周期监测
- 农作物生长状态跟踪
- 农业研究数据分析

## 技术特点
- 采用YOLOv12算法提升检测精度与速度
- 支持实时图像处理与生长阶段识别
- 提供完整的技术实现方案
- 集成数据集处理与用户交互功能
